## 📖 Modo de Uso

### Pré-requisitos

- Node.js 18+
- npm ou yarn
- Git
- Docker (opcional, para produção)

### Instalação Rápida

```bash
# Clone o repositório
git clone https://github.com/guitriloco/project-2.0.git

# Entre no diretório
cd project-2.0

# Instale as dependências do backend
cd backend && npm install

# Instale as dependências do frontend
cd ../frontend && npm install

# Inicie o projeto
cd ../backend && npm run dev
```

### Estrutura de Diretórios

```
project-2.0/
├── frontend/          # React/Next.js frontend
│   ├── src/
│   ├── public/
│   └── package.json
├── backend/           # Node.js backend com Xerebro AI
│   ├── src/
│   ├── services/
│   └── package.json
├── docs/              # Documentação e roadmap
├── infra/             # Infrastructure as Code
└── README.md          # Este arquivo
```

### Configuração

1. **Backend** — O backend já vem configurado com variáveis de ambiente padrão
2. **Frontend** — Conecta-se automaticamente ao backend em `http://localhost:3001`
3. **Xerebro AI** — O cerebro artificial está integrado e ativo

### Desenvolvimento

```bash
# Terminal 1 - Backend
cd backend && npm run dev

# Terminal 2 - Frontend
cd frontend && npm run dev

# Acesse
# Frontend: http://localhost:3000
# Backend API: http://localhost:3001
```
